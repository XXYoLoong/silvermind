# SilverMind 对外介绍文稿

## 一句话介绍

SilverMind 是一套面向适老化认知训练场景的智能互动系统，通过 AI 内容生成、三维空间交互、语音反馈和训练数据记录，把传统认知训练升级为更自然、更沉浸、更容易持续使用的数字化训练体验。

## 项目定位

SilverMind 关注的是老年用户在长期记忆、空间理解、注意力转移和步态控制等方向上的日常训练需求。系统不是把训练做成单纯的题库，而是把视觉、听觉、操作反馈和训练节奏整合在同一个交互界面中，让训练过程更接近日常陪伴式引导。

项目当前包含三个核心模块：

- 长期记忆训练：围绕旧物件、生活场景和年代记忆生成训练内容，配合图片、提示线索和语音播报，引导用户回忆与作答。
- 实体立体迷宫：以三维迷宫模型呈现空间路径，支持路径检测、小球运动演示和即时反馈，强化空间判断与路线理解。
- 方步训练：通过网格路线和计时反馈引导用户完成步态路线训练，兼顾方向切换、注意力控制和运动节奏。

## 技术架构

SilverMind 采用前后端分离架构：

- 前端：React、TypeScript、Vite
- 3D 交互：Three.js、React Three Fiber、Drei
- 状态管理：Zustand
- 后端：Node.js、Express
- AI 接入：后端代理 DeepSeek 文本生成与千问 / DashScope 图片生成
- 数据存储：轻量 JSON 文件存储，便于演示、部署和迁移
- 部署方式：支持本地运行、反向代理部署和云端 API 接入

前端只访问统一的 `/api` 接口，模型密钥始终保存在后端运行环境中。这样的设计避免了把 API Key 暴露给浏览器，也便于在不同项目、不同服务器和不同模型供应商之间迁移。

## 先进性亮点

### 1. AI 驱动的个性化训练内容

传统认知训练常常依赖固定题库，内容容易重复，用户也容易疲劳。SilverMind 通过后端 AI 代理动态生成长期记忆训练题目，可以围绕旧物件、生活用品、时代场景和提示线索形成更有温度的训练内容。

这种方式带来三个优势：

- 内容可持续更新，不依赖人工频繁维护题库。
- 题目更贴近老年用户熟悉的生活经验。
- 可以逐步扩展到不同地区、不同年龄段和不同训练主题。

### 2. 多模态交互，而不是单一答题

SilverMind 不只是“看题、答题、判分”。系统把图片生成、语音播报、提示按钮、动画反馈、路径演示和弹窗反馈组合起来，让用户可以通过多种感官理解训练任务。

对老年用户来说，多模态交互尤其重要。清晰的语音提醒可以降低阅读负担，图片可以帮助唤起记忆，动画可以让反馈更直接，自动消失的结果提示也能减少界面干扰。

### 3. 三维迷宫与真实执行效果结合

迷宫模块不是简单的二维路径图，而是采用 3D 场景呈现空间结构。用户点击检测路径后，系统会先播报检测开始，再让小球沿路径运动，完成后给出路径正确或错误的反馈。

这种设计让训练从“抽象判断”变成“可视化验证”，用户能看到路线是否贯通，也能理解从起点到终点的执行过程。对于空间认知训练、实体迷宫硬件展示和康复训练演示，这种交互比静态图更直观。

### 4. 方步训练兼顾认知与运动节律

方步训练模块通过网格路径、计时、正确率和路线模式引导用户完成步态训练。它不仅记录“走了几步”，还强调方向转换、节奏控制和任务完成度。

在适老化训练中，认知和运动常常是联动的。SilverMind 将路线记忆、方向判断和动作执行放到同一个训练流程中，为后续扩展到康复、日常活动能力训练和长期跟踪打下基础。

### 5. 适老化首屏体验

系统界面优先服务真实训练，而不是做成营销落地页。首屏直接进入训练工具，减少无关说明和跳转，让陪护人员或训练师打开后即可开始操作。

界面设计遵循几个原则：

- 字号、按钮和反馈区域适合快速识别。
- 页面尽量避免因为内容过大而频繁滚动。
- 训练结果、提示和操作按钮保持清晰的视觉层级。
- 弹窗反馈自动消失，也支持点击其它区域关闭。

### 6. 云端 AI 代理更安全

SilverMind 的 AI 调用在后端完成，前端不会保存或打包任何模型密钥。部署时只需要在服务器私有环境变量中配置密钥，浏览器访问的始终是自己的业务 API。

这套架构的好处是：

- API Key 不暴露给前端。
- 可以统一做超时、降级和错误处理。
- 可以平滑替换模型供应商或模型版本。
- 其它项目复用时只需要照着后端代理模式接入。

### 7. 开源友好的工程结构

项目使用木兰宽松许可证第 2 版（Mulan PSL v2）开源，代码结构清晰，前端模块和后端路由相互独立。训练模块、API 调用、语音工具、状态管理和样式文件都有明确边界，便于二次开发和商业化集成。

## 适用场景

SilverMind 可用于：

- 社区养老服务中心的认知训练演示。
- 老年大学、康养机构的互动训练课程。
- 康复训练产品的前端原型或功能模块。
- 智慧养老展厅和产品路演展示。
- AI + 健康管理方向的课程、比赛或开源项目。

## 对机构的价值

对养老机构和康复服务团队来说，SilverMind 的价值不只在于“能训练”，更在于它提供了一套可复制的智能训练产品形态：

- 训练内容可以持续生成，减少内容维护成本。
- 交互形式更生动，提升用户参与度。
- 前后端分离，便于部署到内网、云服务器或演示环境。
- 可继续扩展用户档案、训练报告、风险筛查和硬件联动。
- 开源协议清晰，便于学习、试点和二次开发。

## 后续扩展方向

SilverMind 已经具备 AI 内容生成、多模态反馈、三维交互和训练记录的基础能力，后续可以继续扩展：

- 训练难度自适应：根据历史正确率和用时自动调整题目难度。
- 用户画像：为不同用户维护个性化训练偏好。
- 报告系统：生成阶段性训练趋势和陪护建议。
- 硬件联动：连接实体迷宫、地垫、传感器或康复设备。
- 多模型支持：根据文本、图像、语音任务切换更合适的模型。
- 机构后台：支持多用户、多课程、多训练师管理。

## 合规说明

SilverMind 是认知训练与交互演示系统，不替代医生诊断、临床评估或医疗处方。实际用于医疗、康复或机构服务时，应结合专业人员评估和当地合规要求。

## 宣传短文

SilverMind 让适老化认知训练从固定题库走向智能互动。系统结合 AI 生成、三维空间演示、语音引导和训练数据记录，为长期记忆、空间判断和方步训练提供一体化体验。它既能作为养老机构的互动训练工具，也能作为智慧康养产品的技术原型。通过后端安全代理模型调用，SilverMind 避免前端暴露密钥，并保留了清晰的开源工程结构，适合展示、部署、二次开发和商业化延展。
